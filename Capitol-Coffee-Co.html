Capitol Coffee Co

A potential website design for a coffee shop business

The purpose of this project was to put heavy focus on designing a website for a potential coffee company. During my time working with this project I realized CSS can be quite tricky. There is a lot of tinkering that needs to be done to get things to align the way you want them. To create a responisve website there are a lot of CSS fundamentals such as em, rem, vw, vh, media queries, flex-box, and grid that need to be balanced properly to create a elegant design.

On top of creating a responsive website, I also learned how to create common website features such as a nav bar, shopping cart, and modals for zooming onto items. Finally, for the finishing touches, I tried to pick colors that would be harmonious with each other as well as add transitional effects to add to the overall user experience. Doing this project helped me get exposed to some of the complexities of CSS. It got me excited to do more CSS heavy application as i feel I am improving everytime I have experience with a new CSS-based project.
